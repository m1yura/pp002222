# # PP02 — Проектный практикум (Архитектура и дизайн интерьера)

## Описание

Данный репозиторий содержит материалы проектного практикума ПП.02 по направлению «Программирование в компьютерных системах». В ходе практики выполнялись задачи по проектированию и разработке информационной системы для автоматизации работы архитектурного бюро / дизайн-студии.

Тема: Разработка информационной системы управления проектами в сфере архитектуры и дизайна интерьера.

Практика выполнялась с использованием средств проектирования, моделирования и разработки программного обеспечения.

---

## Содержание репозитория

### /docs
Документация по практике:
- Отчёт по проектному практикуму
- Описание предметной области
- Техническое задание
- UML-диаграммы (6 видов):
  - Use Case — сценарии использования
  - Class — структура классов
  - Sequence — взаимодействие объектов
  - Activity —业务流程
  - Statechart — состояния объектов
  - Component — компонентная архитектура

### /src
Исходный код:
- /backend — серверная часть (Python/Django)
- /frontend — клиентская часть (React)
- /api — спецификация API (OpenAPI/Swagger)

### /diagrams
Исходники UML-диаграмм:
- /plantuml — диаграммы в формате PlantUML
- /drawio — диаграммы в формате Draw.io

### /tests
Тестовые данные и сценарии:
- test_projects.csv — тестовые проекты
- test_clients.csv — тестовые клиенты
- test_scenarios.md — сценарии тестирования

---

## Цель работы

Целью проектного практикума является проектирование и разработка информационной системы для автоматизации деятельности архитектурного бюро / дизайн-студии, позволяющей:
- управлять проектами дизайна интерьера;
- вести клиентскую базу и историю взаимодействий;
- формировать сметы и отслеживать финансовые потоки;
- создавать цифровое портфолио готовых работ;
- автоматизировать рутинные задачи дизайнера.
